
# CrediGuard Frontend

## Overview
This project was developed as part of the PIDEV – 4th Year Engineering Program at Esprit School of Engineering (Academic Year 2025–2026).

CrediGuard Frontend is the client-side application of the project. It provides interfaces for authentication, user management, credit workflows, finance, crowdfunding, and event management.

## Features
- Sign in / Sign up
- Role-based navigation
- Shared user management
- Credit module interfaces
- Finance module interfaces
- Crowdfunding module interfaces
- Event module interfaces

## Tech Stack

### Frontend
- Angular
- TypeScript
- HTML
- CSS

### Backend
- Spring Boot REST API

## Architecture
The frontend is organized into modules, components, services, routing, and guards for role-based access.

2. Install dependencies with npm install
3. Run the project with ng serve
4. Open the application in the browser
